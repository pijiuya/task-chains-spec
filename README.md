# Task Chains Spec

A practical spec and example set for **task chains** — multi-step agent workflows that combine execution, validation, recovery, branching, and auditability.

## Contents
- `docs/task-chains/` — naming, state machine, audit log schema, development guide
- `examples/` — concrete demos exercising linear, recoverable, auditable, and decision-branch chains

## Core preferred term
**闭环决策链 / Closed-Loop Decision Chain**

## Why this repo exists
This repo exists to turn task-chain behavior into a documented, reviewable development pattern instead of leaving it as undocumented prompt behavior.
